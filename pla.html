Lectura de Placas (OCR)

¿Qué es OCR?

OCR significa Optical Character Recognition (Reconocimiento Óptico de Caracteres). Es la tecnología que permite leer textos desde imágenes como fotos, documentos escaneados o placas.

Ejemplo de Código OCR

Aquí tienes un ejemplo básico del proceso OCR:

                import cv2
import pytesseract

# Ruta del ejecutable de Tesseract
pytesseract.pytesseract.tesseract_cmd = r"C:\Program Files\Tesseract-OCR\tesseract.exe"

# 1. Cargar imagen
img = cv2.imread("img_1.png")

# 2. Obtener tamaño
alto, ancho, _ = img.shape

# 3. Zona del recorte (letras grandes)
y1 = int(alto * 0.25)
y2 = int(alto * 0.75)
x1 = int(ancho * 0.15)
x2 = int(ancho * 0.95)
recorte = img[y1:y2, x1:x2]

# --- PREPROCESAMIENTO MEJORADO ---

# Convertir a gris
gris = cv2.cvtColor(recorte, cv2.COLOR_BGR2GRAY)

# Aumentar contraste (truco muy efectivo)
gris = cv2.convertScaleAbs(gris, alpha=2.0, beta=0)

# Eliminar ruido
blur = cv2.GaussianBlur(gris, (5, 5), 0)

# Umbral Otsu (mejor para texto grueso)
_, binaria = cv2.threshold(blur, 0, 255, cv2.THRESH_BINARY_INV + cv2.THRESH_OTSU)

# Aumentar tamaño antes de OCR
binaria_grande = cv2.resize(binaria, None, fx=2, fy=2, interpolation=cv2.INTER_CUBIC)

# --- OCR con whitelist y modo "texto sencillo" ---
config = r'-c tessedit_char_whitelist=ABCDEFGHIJKLMNOPQRSTUVWXYZ0123456789 --psm 7'

texto = pytesseract.image_to_string(binaria_grande, config=config)

# Limpiar texto
placa = texto.strip().replace(" ", "").replace("\n", "")

print("Placa detectada:", placa)

# Mostrar ventanas
cv2.imshow("Recorte", recorte)
cv2.imshow("Binaria", binaria)
cv2.imshow("Binaria Grande", binaria_grande)
cv2.waitKey(0)

Glosario

OCR

Proceso que convierte una imagen con texto en texto editable.

cv2.imread()

Función que carga una imagen desde un archivo.

pytesseract

Librería que hace el reconocimiento de texto.

Grayscale

Imagen convertida a tonos de gris para mejorar lectura OCR.
